## 这份 deck 提出的可核查主张

### A. 平台路线主张

- `NVIDIA` 正把 `CPU + GPU + DPU + switch + context storage` 组织成面向 agentic AI 的一体化平台。
- `AMD` 会沿 `EPYC + Instinct + UALink + ROCm` 走开放平台路线。
- `Intel` 会继续提供 host CPU，并尝试以 `18A + Diamond Rapids + Falcon Shores` 保持平台存在感。

### B. 技术方向主张

- 异构融合会成为主流：`CPU + GPU + NPU + DSA`
- 内存墙突破依赖 `HBM4/5 + CXL + PIM`
- 开放生态会用 `UALink + RISC-V` 对冲封闭平台
- 能效会越来越依赖存内计算、神经形态、专用单元

### C. 数字与时间表主张

- `RISC-V 10% -> 25%`、`ARM 30% -> 35%`、`x86 60% -> 40%`
- `HBM4 2.0-3.3TB/s`、`HBM5 4.0TB/s`
- `AMD 2030 512核`
- `AI chip market 2030 = $295.5B`, `33.2% CAGR`
