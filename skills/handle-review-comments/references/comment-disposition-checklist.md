# Comment Disposition Checklist

Use this checklist after processing review comments.

## Coverage

- every comment block has a disposition
- no reviewer bullet is left hanging without a response
- cross-chapter comments identify all affected files

## Quality of Judgment

- `接受` items are actually reflected in正文
- `部分接受` items explain the retained boundary
- `不接受新增修改` items give a concrete reason, not a vague refusal

## Draft Integrity

- no正文 remains after `参考文献`
- accepted fixes did not duplicate nearby chapters
- transitions still make sense after restructuring
- reused figures now have chapter-specific captions where needed

## Rebuild

- `rebuilt.md` is regenerated if chapter files changed
- rebuilt figure paths still resolve from the combined draft location
