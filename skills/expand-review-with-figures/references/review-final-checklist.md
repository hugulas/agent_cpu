# Review Final Checklist

Use this checklist before declaring a综述稿 complete.

## Structure

- `manifest.md` order matches the rebuilt draft
- each chapter has one clear purpose
- no duplicated intro blocks or leftover draft scaffolding
- no正文 remains after `参考文献`
- transitions between adjacent chapters are explicit

## Evidence

- every major judgment has nearby citation support
- every important number is sourced
- reused numbers are not silently drifting across chapters
- direct evidence and inference are distinguishable in wording
- weak industrial signals are not overstated as hard proof

## Figures and Tables

- each figure supports a named judgment
- each figure caption states why it matters in this chapter
- reused figures have distinct captions across chapters
- visually dense chapters explain the figure instead of stacking images
- thin chapters get either one useful figure or one compact evidence table

## Rebuild and Paths

- `rebuilt.md` is regenerated after edits
- figure paths work from the rebuilt file location
- cross-workspace or shared assets use correct relative paths

## Delivery

- the rebuilt draft reads like one report rather than pasted chapter fragments
- key claims close with implication, not just observation
- terminology is stable across chapters
- residual gaps are explicit if evidence is still incomplete
